Defining an ODE Problem

To define an ODE Problem, you simply need to give the function $f$ and the initial condition $u₀$ which define an ODE

\[\frac{du}{dt} = f(t,u)\]

f should be specified as f(t,u) and u₀ should be an AbstractArray whose geometry matches the desired geometry of u. Note that we are not limited to numbers or vectors for u₀, one is allowed to provide u₀ as arbitrary matrices / higher dimension tensors as well.

Problem Type

DifferentialEquations.ODEProblem — Type.

ODEProblem

Wraps the data which defines an SDE problem

\[\frac{du}{dt} = f(u,t)\]

with initial condition $u₀$.

Constructors

ODEProblem(f,u₀;analytic=nothing) : Defines the SDE with the specified functions and defines the solution if analytic is given.

Fields

  • f: The drift function in the ODE.

  • u₀: The initial condition.

  • analytic: A function which describes the solution.

  • knownanalytic: True if the solution is given.

  • numvars: The number of variables in the system
